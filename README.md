# Mihsan Alam — Portfolio

Personal portfolio website showcasing my experience, skills, and projects as a Full Stack Engineer.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Theme Support**: [next-themes](https://github.com/pacocoursey/next-themes)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Features

- **Dynamic Hero Section**: Interactive taglines with typing animation.
- **Dark/Light Mode**: Smooth theme toggling (Dark mode default).
- **Responsive Layout**: Designed for mobile, tablet, and desktop screens.
- **Projects Grid**: Categorized portfolio filter with interactive details modal and image gallery.
- **Experience Timeline**: Vertical timeline illustrating career milestones.
- **Direct Contact**: Fast direct-contact buttons and location status.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
